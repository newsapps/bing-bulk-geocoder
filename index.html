Tribune Geocoder

Instructions

Create a CSV file containing the addresses you want to geocode. 1 column should contain an ID field, 1 column should contain an address, and there should be 1 column each for Latitude and Longitude.

Make sure to use the exact names for your columns, as in the example below:

Id
GeocodeRequest/Query
GeocodeResponse/Point/Latitude
GeocodeResponse/Point/Longitude
